# G-code Output Directory

This directory will contain the generated G-code files from slicing.

Files will be named:
- `{model-name}_a1.gcode` - From simple API
- `{model-name}_a1_builder.gcode` - From builder API

You can then:
1. Transfer these to your A1 printer via SD card or network
2. Compare with G-code generated by BambuStudio GUI
3. Analyze the G-code to verify settings
